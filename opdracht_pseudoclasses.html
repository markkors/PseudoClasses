CSS / HTML Pseudo Classes

leerdoel:

Je leert in deze oefening hoe je via CSS verschillende pseudo classes gebruikt.

Opdracht:

Volg de omschrijving op bij ieder element in de onderstaande code

  • Je mag de CSS toevoegen aan de externe stylesheet css/styles.css
  • Eventueel kun je de styling toevoegen binnen de HTML binnen de <style> tags in de <head>
  • Het kan nodig zijn om in de HTML elementen class attributen toe te voegen en deze te gebruiken in je stylesheet
